## Файли:
- `index.html` – Основна HTML-сторінка
- `styles.css` – Стилі сайту
- `script.js` – Скрипти
- `IMG_1260.jpeg` – Обкладинка книги
- `IMG_1262.jpeg` – Фон сайту
